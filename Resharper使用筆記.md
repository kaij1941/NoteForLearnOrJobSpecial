# Resharper 使用筆記
<h2> 快捷鍵 </h2>
1. 擷取函式 		: Ctrl + Alt + "M" 							<br>
2. 進入實作 		: F12 / Alt + End(有介面時)					<br>
3. 整段程式碼移動	: Ctrl + Shift + Alt + 方向鍵				<br>
4. 搜尋 文字/檔案   : Ctrl + "T"								<br>		
5. 當前文件中查詢	: Alt  + "\"								<br>						
6. ToDoExplore      : Ctrl + Alt + "." 							<br>
7. 顯示多型         : Ctrl + "P"								<br>
8. 更名             : (Ctrl +"R" )*2                            <br>
9. 選取區塊         : Ctrl + Shift  + 方向鍵(左-/右+)				<br>
10. 提取方法        : Ctrl + "R" 及 Ctrl +  "M"        <br>
